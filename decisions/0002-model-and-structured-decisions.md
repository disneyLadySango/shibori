# ADR-0002: 言語モデル判断と構造化出力

- Status: accepted
- Date: 2026-07-20

## Context

Build WeekではGPT-5.6の利用が外部制約である。配分、理解評価、学習経路の判断結果を射影側が安全に扱うには、期待する形と実際の出力のずれを減らす必要がある。

## Decision

- すべての言語モデル判断にGPT-5.6を使用する。
- 機械的に利用する判断はstrict structured outputsで受け取る。
- 同じ配分判断から派生する結果は、整合性を優先して可能な範囲で一回の判断へまとめる。

## Consequences

- 外部能力の可用性、遅延、利用費の影響を受ける。
- スキーマ適合は保証できても、教育判断の正しさは第4層で別途検証する必要がある。
- デモ結果と実判断を明確に区別する必要がある。
