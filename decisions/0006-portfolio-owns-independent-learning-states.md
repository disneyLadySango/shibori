# ADR-0006: ポートフォリオによる学習状態の分離

- Status: accepted
- Date: 2026-07-21

## Context

複数の学習目的が一つの学習状態を共有すると、片方の理解確認や抜けが他方へ混入する。目的間のおすすめをそのまま選択へ適用すると、学習者の最終決定も失われる。

## Decision

- バージョン付き学習ポートフォリオが、目的ごとに完全な学習状態を所有する。
- 一つの目的の更新は、識別子が一致する学習状態だけを置き換える。
- 目的間のおすすめと学習者が選んだ目的を別の状態として保持する。
- 既存の単一目的状態は、内容を変えず一目的のポートフォリオへ移行する。

## Consequences

- 目的の切替だけでは、現在地、理解状態、確認結果、抜けを失わない。
- おすすめの生成に失敗しても、学習者の選択は変わらない。
- 将来、保留・終了・配分履歴を追加する場合は、ポートフォリオのバージョン移行が必要になる。
