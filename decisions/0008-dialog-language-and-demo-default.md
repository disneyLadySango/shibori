# ADR-0008: 対話言語と審査用初期表示

- status: accepted
- date: 2026-07-22

## Context

Shiboriは日本語の体験として始まったが、国際的な審査者が再構築なしで試せる公開デモを提出する。対話言語の変更は学習内容や理解状態の変更ではない。

## Decision

- 対応する対話言語は英語と日本語とする。
- 公開デモの初期対話言語は英語とし、常に日本語へ切り替えられるようにする。
- 選択した対話言語は学習ポートフォリオとは別に端末へ保持する。
- 言語モデルへ判断を依頼する際は対話言語を明示し、音声の話し方も同じ言語へ合わせる。
- 学習状態の識別子と内部状態値は対話言語から独立させる。

## Consequences

- 審査者は初回から英語で主要動線を試せる。
- 学習者は言語変更後も同じ学習状態を継続できる。
- 新しい利用者向け文言とデモ生成文には英語・日本語の双方が必要になる。
