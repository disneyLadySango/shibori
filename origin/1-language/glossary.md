# ユビキタス言語 — 初期ドラフト

状態: **人間承認待ち**

下位層は、重要概念についてこの文書の語彙だけを使う。

| 用語 | 定義 | 含めないもの |
| --- | --- | --- |
| 学習者 | 仕事や生活上の責務と並行して、自分の目的のために学ぶ人 | 学校種別や年齢だけで区切った属性 |
| 学習目的 | 学びによって可能にしたい、学習者自身の現実の変化 | 教材を終えること自体 |
| 学習領域 | 一つの学習目的に向けて知識と実践を積み上げるまとまり | 単なるコンテンツ分類 |
| 学習経路 | 学習目的までに必要な理解と実践の依存関係、および現在地 | 固定された教科書の目次 |
| 集中資源 | 学習者が一定期間に使える、注意の深さと時間の組み合わせ | 総可処分時間だけ |
| 深い集中 | 計算、記述、判断、想起など、能動的に手を動かす注意状態 | 長時間であること自体 |
| 軽い集中 | 移動や家事と並行しながら概念や関係をつかめる注意状態 | 理解を確認せず聞き流すこと |
| 集中先 | 現在の集中資源を優先的に投じる、ただ一つの理解または実践 | 興味のある候補一覧 |
| 配分 | 学習目的、現在地、集中資源に基づき、何へどの深さの注意を使うか決めること | 教材を均等に消化すること |
| 理解証拠 | 学習者が説明、計算、判断、想起によって示した観測可能な理解 | 閲覧時間や再生完了だけ |
| 抜け | 先へ進むために必要だが、理解証拠が不足している前提 | 単なる苦手意識 |
| 補強 | 抜けを解消するため、一時的に学習経路の優先順位を変えること | 失敗への罰や全体のやり直し |
| 再配分 | 新しい理解証拠または抜けに応じ、集中先と学習経路を更新すること | 毎回すべてを作り直すこと |
| 学習ポートフォリオ | 学習者が並行して持つ学習領域と、それらへの集中資源の配分全体 | 成績や資格の一覧 |
| 今日 | 学習者が次に利用できる一回の軽い集中または深い集中の機会 | 暦日の0時から24時に限定しない |

## 追加提案が必要な語

- 「習得」を、自己申告・単一の正答・複数回の理解証拠のどれで成立させるかは未決定。
- 「撤退」を、学習領域の停止とプロダクト仮説の停止のどちらにも使うと曖昧になるため、承認時に分離を検討する。
