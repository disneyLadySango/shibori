# TS-001 学習目的だけで探索を始める

1. 学習者が、期限や実用性を伴わない「知りたい」という学習目的だけを述べる。
2. その学習目的が成立し、優先度を下げられず、興味を具体化する探索が一つ推奨されることを確認する。
3. 到達状態を決めずに探索を始められることを確認する。
4. 優先度が自動的に下げられていないことを確認する。
