# TS-001 状況変化後の再評価
- state: `active`
- revision: `2`
- story_id: `US-012`
- ac_id: `AC-001`
1. 複数の学習目的について、現在の状況に基づく一つのおすすめを受ける。
2. 一つの学習目的の期限、重要度、利用場面のいずれかを変更する。
3. 変更前後のおすすめと、判断に使われた変化を区別して説明できることを確認する。
4. 各学習目的の到達状態、現在地、理解状態、確認結果、抜けが変化していないことを確認する。
