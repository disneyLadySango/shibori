# TS-001 学習目的の休止と再開
- state: `active`
- revision: `2`
- story_id: `US-013`
- ac_id: `AC-001`
1. 進行中の学習目的について、到達状態、現在地、理解状態、確認結果、抜けを確認する。
2. その学習目的を休止し、次のおすすめ候補から外れることを確認する。
3. 休止した学習目的を再開する。
4. 休止前と同じ到達状態、現在地、理解状態、確認結果、抜けから続けられることを確認する。
