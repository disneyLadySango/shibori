# TS-001 学習目的と到達状態の区別

- state: `backlog`
- story_id: `US-001`
- ac_id: `AC-001`
- approach_ref: 未設定

1. 学習者が、期限や実用性を伴わない「知りたい」という学習目的だけを述べる。
2. その学習目的が成立し、優先度を下げられず、興味を具体化する探索が一つ推奨されることを確認する。
3. 到達状態を決めずに探索を始められることを確認する。
4. 探索後、「理解したい」というWillは到達状態として確定せず、「説明できる」などのCanを後から作れることを確認する。
5. 到達状態が定まった後は、学習目的と到達状態が区別され、次の集中判断の理由として使われることを確認する。
